Anime Planet to AniList:

  1. Export Anime Planet data.
  2. Download the anitransfer repo.
  3. Put Anime Planet JSON file in repo folder.
  4. Change anitransfer.py to target the file.
  5. Run anitransfer.py script.
  6. Confirm titles that need manual review.
  7. Check folder for finalized XML file.
  8. Import file as MyAnimeList XML.
  9. Send me your "cache.csv" file to improve the script.
